# Student Marketplace Project

## About
Final project for CSC 2053 Platform Based Computing.
This is a simple marketplace app for Villanova students, inspired by Facebook Marketplace.
Students can browse items for sale on the home page, save their favorites, and list items on the account page.
Created using Angular and Ionic.
